    Eclipse IDE

    Skip to end of metadata
    • Created by Paul-Henri Giraud, last modified by Sylvain Hitier on Apr 18, 2014
    Go to start of metadata

    Customizing window title

    Append the filesystem location of the workspace

    Add -showlocation in eclipse.ini file, before -vmargs option.

    See http://wiki.eclipse.org/Eclipse.ini

     

    Prepend any text, typically a nickname for the workspace

    1. «Window» menu / «Preferences» item; «Preferences» modal window
      1. in LHS tree, «General» / «Workspace»
      2. in RHS, «Workspace name (shown in window title):»

    Set up custom dictionary

    This helps avoiding typos in comments!

    1. «Window» menu / «Preferences» item; «Preferences» modal window
      1. in LHS tree, «General» / «Editors» / «Text Editors» / «Spelling»
      2. in RHS, in «Dictionaries» zone, in «User defined dictionary» textbox:
        1. \\srv-aix-share\Echanges\eclipse_dictionary\eclipse_dictionary.txt
